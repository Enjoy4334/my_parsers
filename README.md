## Installation

  
  Clone the Repository: 

```sh
https://github.com/codeberrypro/my_parsers

```


  Activate : 

```sh
python -m venv venv
source venv/bin/activate (Linux/Mac)
venv\Scripts\activate (Windows)

```

  Install poetry: 

```sh
curl -sSL https://install.python-poetry.org | python3 -
```
